# HandPad

Project page for **HandPad: A Bimanual Hand Interface for Fluid Window Interactions in VR**.

Published at [https://handpad.github.io](https://handpad.github.io).

## Publish

1. On GitHub, rename this repository from `Project` to `handpad.github.io`.
2. In **Settings → Pages**, deploy from the `main` branch at `/ (root)`.
3. Push this repo:

```bash
cd /Users/yingw/Desktop/handpad.github.io
git remote add origin git@github.com:HandPad/handpad.github.io.git
git push -u origin main
```

If the repo is still named `Project`, use `HandPad/Project.git` for the first push, then rename on GitHub.
